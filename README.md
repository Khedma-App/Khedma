## 📌 Overview
**Khedma** is a scalable, cross-platform smart service platform designed to seamlessly connect users with professional service providers. 

🏆 **Funded Project:** This project was awarded a **60,000 EGP development grant** by the Academy of Scientific Research and Technology (ASRT) in Egypt to accelerate its market launch.

## ✨ Key Features
* **User & Provider Portals:** Distinct, optimized experiences for both customers and service providers.
* **Pixel-Perfect UI:** Fully responsive and intuitive user interfaces implemented strictly from Figma designs.
* **Real-time Data:** Seamless backend integration for real-time updates, authentication, and database management.
* **Scalable Architecture:** Built from the ground up to support future growth and feature expansion.

## 🛠️ Tech Stack & Architecture
* **Framework:** Flutter / Dart
* **Architecture:** Clean Architecture (Feature-First approach) to ensure separation of concerns, testability, and maintainability.
* **State Management:** Cubit / BLoC
* **Backend & BaaS:** Firebase & Supabase
* **Design Implementation:** Figma to Flutter (Pixel-perfect & Responsive)
